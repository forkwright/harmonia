# Code of conduct

## Standards

**Expected behavior**:
- Respectful and constructive communication
- Focus on technical merit
- Assume good intent
- Provide helpful feedback
- Be patient with questions

**Unacceptable behavior**:
- Harassment, discrimination, or personal attacks
- Trolling or inflammatory comments
- Publishing private information
- Off-topic or spam content

## Scope

Applies to all project spaces: GitHub issues, PRs, discussions, code comments, and any official communication channels.

## Enforcement

Violations will be addressed by project maintainers. Actions may include warnings, temporary bans, or permanent removal from the project.

## Reporting

Report violations via GitHub or email project maintainers directly.
